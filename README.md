# 米兔分帳小幫手 V1.2.4

旅行用分帳 PWA。重點是「誰先付、誰分攤、最後誰轉給誰」，不是完整會計系統。

## V1.1 新增功能

- 行程分帳：不同旅行可分開記錄，例如大邱三日行、廣島遊。
- 現金池：可記錄誰先放錢進共同錢包，支出可選現金池支付。
- 原幣別結算：台幣、日圓、韓幣、美金分開結算，不使用當日匯率。
- 雲端收據／票券連結：支出可附連結，也可另外建立連結庫。
- 日期搜尋：用日期下拉選單篩支出，也可再加關鍵字搜尋。
- JSON 備份：可匯出全部行程，也可只匯出目前行程。

## 使用方式

1. 開啟 `index.html` 測試。
2. 若要安裝成 PWA，請將整個資料夾放到 GitHub Pages、Netlify 或其他 HTTPS 網站。
3. 用手機瀏覽器開啟後，選擇「加到主畫面」。

## 注意

- 資料存在瀏覽器本機，清除瀏覽資料前請先匯出 JSON。
- 現金池各幣別分開，不會互相折算。
- 如果現金池支出超過已登錄出資，結算頁會提示補登。
